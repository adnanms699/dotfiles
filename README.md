My personal macOS setup managed with Homebrew.

## Clone

```bash
git clone https://github.com/adnanms699/dotfiles.git
```

## What is Homebrew?

[Homebrew](https://brew.sh) is a package manager for macOS. It lets you install, update, and manage software from the command line — no more manual downloads and drag-and-drop installs.
